# Kiro 监控方案流程图

## 方案 1: 定时器方案（推荐）

```
┌─────────────────────────────────────────────────────────────┐
│                    启动定时器脚本                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│                    设置参数                                  │
│  - 间隔: 5 分钟                                              │
│  - 最大次数: 100 次                                          │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────────────┐
                    │  开始循环      │
                    └───────────────┘
                            ↓
                    ┌───────────────┐
                    │  等待 5 分钟   │
                    └───────────────┘
                            ↓
                    ┌───────────────┐
                    │ 查找 Kiro 窗口 │
                    └───────────────┘
                            ↓
                ┌───────────┴───────────┐
                │                       │
            找到了？                 没找到
                │                       │
                ↓                       ↓
        ┌───────────────┐       ┌───────────────┐
        │ 检查计数       │       │ 跳过本次       │
        └───────────────┘       └───────────────┘
                ↓                       │
        ┌───────────┴───────────┐       │
        │                       │       │
    达到最大次数？           没达到      │
        │                       │       │
        ↓                       ↓       │
┌───────────────┐       ┌───────────────┐
│ 停止监控       │       │ 发送 "continue"│
└───────────────┘       └───────────────┘
                                ↓
                        ┌───────────────┐
                        │ 计数 +1        │
                        └───────────────┘
                                ↓
                        ┌───────────────┐
                        │ 保存计数       │
                        └───────────────┘
                                ↓
                        ┌───────────────┐
                        │ 回到循环开始   │
                        └───────────────┘
```

**特点**:
- ✅ 不需要检测任何状态
- ✅ 定期发送，不会漏掉
- ✅ 简单可靠

## 发送 "continue" 流程

```
┌─────────────────────────────────────┐
│         需要发送 "continue"          │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  1. 复制 "continue" 到剪贴板         │
│     Clipboard.SetText("continue")    │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  2. 激活 Kiro 窗口                   │
│     ShowWindow(handle, 9)            │
│     SetForegroundWindow(handle)      │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  3. 等待窗口激活                     │
│     Sleep 500ms                      │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  4. 模拟粘贴（Ctrl+V）               │
│     SendKeys.SendWait("^v")          │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  5. 等待粘贴完成                     │
│     Sleep 200ms                      │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│  6. 模拟回车                         │
│     SendKeys.SendWait("{ENTER}")     │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│         发送完成                     │
└─────────────────────────────────────┘
```

## 两种方案对比

### 定时器方案

```
简单直接：
    等待 → 发送 → 等待 → 发送 → ...
    
优势：
    ✅ 不需要检测
    ✅ 不会漏掉
    ✅ 简单可靠
    
代价：
    ⚠️ 可能多余发送
```

### 智能监控方案

```
复杂判断：
    检查标题 → 对比 → 计算时间 → 判断 → 发送
    
优势：
    ✅ 只在需要时发送
    ✅ 可以看到状态
    
代价：
    ⚠️ 可能误判
    ⚠️ 可能漏判
    ⚠️ 逻辑复杂
```
